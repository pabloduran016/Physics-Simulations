# Physics-Simulations
Repo containing various physics simulations in python

## Getting Started
```console
pip install -r requirements.txt
```

## Waves 3d  
**Run:**
(It needs to be run as a module (`-m` flag) to be able to access the [`colors.py`](colors.py) file 
```console
python -m waves3d
```
